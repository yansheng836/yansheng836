### starchart.cc

> 现在，你只需要在仓库url中把 `github.com` 域名改为 `starchart.cc` 回车即可观看 Star 趋势了
>  而且，还支持把该趋势图插入到自己仓库的 Readme.md 文件中哦！
>  当然，由于是 Markdown 插图语法，也可以插入到自己的博客(简书不行，不支持外链图片且转站内还失败)。
